1. はじめに

三賢会議（以下「本アプリ」）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、本アプリがどのような情報を収集し、どのように使用・保護するかについて説明します。

本アプリを使用することで、本プライバシーポリシーに同意したものとみなされます。

2. 収集する情報

2.1 ユーザーが提供する情報

審議テーマ・相談内容

  • ユーザーが入力する審議テーマや相談内容
  • 審議履歴データ
  • 保存した決議書

使用目的:

  • AI賢者による審議の実行
  • 審議履歴の保存と表示
  • サービス品質の向上

アカウント情報（オプション）

ユーザーがログイン機能を使用する場合:

  • Firebase Authenticationを通じた認証情報
  • メールアドレス（使用する認証方法による）

使用目的:

  • ユーザー認証
  • データの同期とバックアップ

2.2 自動的に収集される情報

デバイス情報

  • デバイスの種類（iPhone、iPadのモデル）
  • OS バージョン（iOS/iPadOSのバージョン）
  • アプリのバージョン
  • 言語設定
  • タイムゾーン

使用状況データ

  • アプリの起動回数
  • 機能の使用状況
  • 審議の実行回数
  • エラーログとクラッシュレポート

広告関連情報

  • 広告識別子（IDFA: Identifier for Advertisers）
  • 広告の表示回数とクリック数
  • 広告のパフォーマンスデータ

3. 第三者サービスの利用

本アプリは、以下の第三者サービスを利用しています:

3.1 Firebase (Google)

  • Firebase Authentication: ユーザー認証、データ同期
  • Firebase Analytics: 利用分析、サービス改善
  • Firebase Crashlytics: アプリの安定性向上、バグ修正

Firebase プライバシーポリシー

3.2 Google AdMob

パーソナライズド広告の配信と広告効果測定に使用します。

  • Google プライバシーポリシー
  • 広告設定

パーソナライズド広告のオプトアウト

iOSの「設定」アプリから広告トラッキングを制限できます:

  1. 設定 > プライバシーとセキュリティ > トラッキング
  2. 「Appからのトラッキング要求を許可」をオフ

3.3 OpenAI API

AI審議機能に使用します。ユーザーが入力した審議テーマ・相談内容のみが送信されます。

重要な注意事項:

  • OpenAIにはユーザー個人を特定できる情報は送信されません
  • 審議内容のみが処理のために送信され、30日後に削除されます
  • OpenAIは送信されたデータをモデルのトレーニングには使用しません

OpenAI プライバシーポリシー

4. データの保存と保護

4.1 データの保存場所

  • ローカルストレージ: デバイス内（Hive データベース）
  • クラウドストレージ: Firebase（ログイン時）
  • サーバー: バックエンドAPI（一時的な処理のみ）

4.2 セキュリティ対策

  • HTTPS通信による暗号化
  • Firebase Authenticationによる認証
  • 定期的なセキュリティアップデート

4.3 データの保持期間

  • 審議履歴: ユーザーが削除するまで保持
  • クラッシュレポート: 90日間
  • 分析データ: 14ヶ月間（Firebaseのデフォルト設定）
  • OpenAI処理データ: 30日間（OpenAI API ポリシー）

5. ユーザーの権利

5.1 アクセス権

保存されているご自身のデータにアクセスする権利があります。

5.2 修正権

不正確なデータの修正を要求する権利があります。

5.3 削除権（忘れられる権利）

以下の方法でデータを削除できます:

  • アプリ内の履歴画面から個別削除
  • アカウント削除（全データ削除）
  • サポートへの削除依頼: support@sankenkaigi.com

5.4 トラッキング拒否権

広告トラッキングを拒否する権利があります（iOS設定から）。

6. 子供のプライバシー

本アプリは13歳未満の子供を対象としていません。13歳未満の子供から意図的に個人情報を収集することはありません。

保護者の方へ: お子様が個人情報を提供したと思われる場合は、support@sankenkaigi.com までご連絡ください。速やかに削除いたします。

7. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービスの改善に伴い、予告なく変更される場合があります。

重要な変更がある場合は:

  • アプリ内通知
  • メール通知（登録ユーザーのみ）
  • 本ページでの告知

8. お問い合わせ

プライバシーに関するご質問、ご懸念、データ削除のご要望は、以下までご連絡ください:

Email: support@sankenkaigi.com

件名: 「プライバシーに関するお問い合わせ」

返信時間: 通常1-3営業日以内に返信いたします。

9. 同意の取得

本アプリの初回起動時に、以下について同意を求めます:

必須の同意

  • 本プライバシーポリシー
  • 利用規約

オプションの同意

  • App トラッキングの許可
    • 目的: パーソナライズド広告の配信
    • 拒否した場合: 非パーソナライズド広告が表示されます
    • 後から変更: iOS設定 > プライバシーとセキュリティ > トラッキング
