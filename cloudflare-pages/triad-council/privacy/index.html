プライバシーポリシー

三賢会議（Triad Council）

最終更新日: 2026年1月31日

三賢会議（以下「本アプリ」）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、本アプリがどのような情報を収集し、どのように使用・保護するかについて説明します。

本アプリを使用することで、本プライバシーポリシーに同意したものとみなされます。

1. 収集する情報

1.1 ユーザーが提供する情報

審議テーマ・相談内容

  • ユーザーが入力する審議テーマや相談内容
  • 審議履歴データ
  • 保存した決議書

使用目的:

  • AI賢者による審議の実行
  • 審議履歴の保存と表示
  • サービス品質の向上

アカウント情報（オプション）

ユーザーがログイン機能を使用する場合:

  • Firebase Authenticationを通じた認証情報
  • メールアドレス（使用する認証方法による）

1.2 自動的に収集される情報

デバイス情報

  • デバイスの種類（iPhone、iPadのモデル）
  • OS バージョン（iOS/iPadOSのバージョン）
  • アプリのバージョン
  • 言語設定
  • タイムゾーン

使用状況データ

  • アプリの起動回数
  • 機能の使用状況
  • 審議の実行回数
  • エラーログとクラッシュレポート

広告関連情報

  • 広告識別子（IDFA: Identifier for Advertisers）
  • 広告の表示回数とクリック数
  • 広告のパフォーマンスデータ

2. 情報の使用目的

収集した情報は、以下の目的で使用されます:

2.1 サービスの提供

  • AI審議機能の実行
  • 審議履歴の保存と表示
  • 決議書の生成と共有

2.2 サービスの改善

  • アプリの不具合修正
  • 新機能の開発
  • ユーザーエクスペリエンスの向上

2.3 広告配信

  • 適切な広告の表示
  • 広告効果の測定と最適化

2.4 分析と統計

  • 利用統計の作成（匿名化データ）
  • サービス改善のための分析

3. 第三者サービスの利用

本アプリは、以下の第三者サービスを利用しています:

3.1 Firebase (Google)

Firebase Authentication

  • 収集する情報: 認証情報、メールアドレス（認証方法による）
  • 使用目的: ユーザー認証、データ同期
  • プライバシーポリシー: firebase.google.com/support/privacy

Firebase Analytics

  • 収集する情報: アプリ使用状況、デバイス情報
  • 使用目的: 利用分析、サービス改善
  • プライバシーポリシー: firebase.google.com/support/privacy

Firebase Crashlytics

  • 収集する情報: クラッシュレポート、デバイス情報、エラーログ
  • 使用目的: アプリの安定性向上、バグ修正
  • プライバシーポリシー: firebase.google.com/support/privacy

3.2 Google AdMob

広告配信

  • 収集する情報: 広告識別子(IDFA)、デバイス情報、位置情報（概略）、閲覧履歴
  • 使用目的: パーソナライズド広告の配信、広告効果測定
  • プライバシーポリシー: policies.google.com/privacy
  • 広告設定: adssettings.google.com

パーソナライズド広告のオプトアウト

iOSの「設定」アプリから広告トラッキングを制限できます:

  1. 設定 > プライバシーとセキュリティ > トラッキング
  2. 「Appからのトラッキング要求を許可」をオフ

3.3 OpenAI API

AI審議機能

  • 収集する情報: ユーザーが入力した審議テーマ・相談内容
  • 使用目的: AI賢者による審議の実行
  • プライバシーポリシー: openai.com/privacy
  • データ保持: OpenAIのAPIポリシーに準拠（通常30日間でデータ削除）

重要な注意事項:

  • OpenAIにはユーザー個人を特定できる情報は送信されません
  • 審議内容のみが処理のために送信され、30日後に削除されます
  • OpenAIは送信されたデータをモデルのトレーニングには使用しません

4. データの保存と保護

4.1 データの保存場所

  • ローカルストレージ: デバイス内（Hive データベース）
  • クラウドストレージ: Firebase（ログイン時）
  • サーバー: バックエンドAPI（一時的な処理のみ）

4.2 セキュリティ対策

  • HTTPS通信による暗号化
  • Firebase Authenticationによる認証
  • 定期的なセキュリティアップデート

4.3 データの保持期間

  • 審議履歴: ユーザーが削除するまで保持
  • クラッシュレポート: 90日間
  • 分析データ: 14ヶ月間（Firebaseのデフォルト設定）
  • OpenAI処理データ: 30日間（OpenAI API ポリシー）

5. ユーザーの権利

5.1 アクセス権

保存されているご自身のデータにアクセスする権利があります。

5.2 修正権

不正確なデータの修正を要求する権利があります。

5.3 削除権（忘れられる権利）

以下の方法でデータを削除できます:

  • アプリ内の履歴画面から個別削除
  • アカウント削除（全データ削除）
  • サポートへの削除依頼: marumi.works@gmail.com

5.4 異議申し立て権

データ処理に異議を申し立てる権利があります。

5.5 データポータビリティ権

ご自身のデータのエクスポートを要求できます。

5.6 トラッキング拒否権

広告トラッキングを拒否する権利があります（iOS設定から）。

6. 子供のプライバシー

本アプリは13歳未満の子供を対象としていません。13歳未満の子供から意図的に個人情報を収集することはありません。

保護者の方へ: お子様が個人情報を提供したと思われる場合は、marumi.works@gmail.com までご連絡ください。速やかに削除いたします。

7. 国際的なデータ転送

本アプリは以下の地域のサーバーを使用する可能性があります:

  • 米国（Firebase、OpenAI）
  • 日本（バックエンドAPI）

これらの地域へのデータ転送は、適切なセキュリティ対策のもとで行われます。

8. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービスの改善に伴い、予告なく変更される場合があります。

重要な変更がある場合は:

  • アプリ内通知
  • メール通知（登録ユーザーのみ）
  • 本ページでの告知

変更後も継続してアプリを使用することで、変更後のプライバシーポリシーに同意したものとみなされます。
