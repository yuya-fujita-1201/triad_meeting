# テスト用相談事項スクリプト

## 基本機能テスト

### 1. シンプルな相談
```
朝早く起きられるようになりたいです。どうすればいいですか？
```

### 2. ビジネス系の相談
```
新しいスタートアップを始めようと思っています。初期段階で最も重要なことは何でしょうか？
```

### 3. キャリア相談
```
現在のエンジニアの仕事にやりがいを感じなくなってきました。転職すべきか、今の会社で部署異動を希望すべきか迷っています。
```

### 4. 技術的な相談
```
FlutterアプリでFirebaseを使った認証とデータ同期を実装したいのですが、ベストプラクティスを教えてください。
```

### 5. 人間関係の相談
```
職場の同僚とコミュニケーションがうまくいきません。意見が対立することが多く、プロジェクトの進行に支障が出ています。
```

## エッジケーステスト

### 6. 非常に短い相談
```
転職したい
```

### 7. 長文の相談（300文字以上）
```
現在30歳で、大手IT企業でソフトウェアエンジニアとして働いています。入社して8年が経ち、技術的なスキルは十分に身につきましたが、最近マネジメント業務が増えてきて、コーディングする時間が減ってきました。一方で、技術の最前線で働き続けたいという思いも強く、管理職になることに抵抗を感じています。会社としては私をマネージャー候補として育成したいようで、来年度からチームリーダーのポジションを打診されています。家族もいるので収入面も重要ですし、キャリアパスとして間違った選択はしたくありません。技術者として深掘りするか、マネジメントの道を進むか、どちらを選ぶべきでしょうか？
```

### 8. 曖昧な相談
```
最近なんだかモヤモヤします
```

### 9. 複数の質問を含む相談
```
副業を始めたいと思っています。どんな副業がおすすめですか？また、本業との両立のコツや、税金の申告方法についても教えてください。
```

### 10. 専門的な相談
```
機械学習モデルのプロダクション環境へのデプロイについて悩んでいます。MLOpsのベストプラクティスと、モデルのバージョン管理、モニタリング方法について知りたいです。
```

## カテゴリ別テスト

### 11. 健康・ライフスタイル
```
在宅勤務で運動不足になり、体重が5kg増えました。効果的なダイエット方法と運動習慣の作り方を教えてください。
```

### 12. 学習・スキルアップ
```
プログラミング初心者です。Web開発を学びたいのですが、何から始めればいいでしょうか？おすすめの学習ロードマップを教えてください。
```

### 13. お金・投資
```
30代から資産運用を始めたいと思っています。投資信託やNISAについて、初心者向けのアドバイスをください。
```

### 14. 起業・独立
```
フリーランスエンジニアとして独立を考えています。会社員を辞める前に準備しておくべきことを教えてください。
```

### 15. 家族・育児
```
共働きで子どもが2歳です。仕事と育児の両立に疲れています。時間管理とストレス解消のコツを教えてください。
```

## ストレステスト

### 16. 記号を含む相談
```
API設計で悩んでいます！RESTful vs GraphQL、どちらを選ぶべき？？プロジェクトは中規模で、モバイル+Web対応です。性能・保守性・開発速度の観点から教えてください！！
```

### 17. 改行を含む相談
```
次の3点について相談があります：

1. リモートワークでの生産性向上
2. オンラインでのチームビルディング
3. 時差のあるチームとのコミュニケーション

具体的なアドバイスをください。
```

### 18. 数字を含む相談
```
2025年に転職市場はどうなると思いますか？35歳で年収700万円、転職で年収アップは可能でしょうか？
```

### 19. 英語を含む相談
```
海外のスタートアップでSenior Software Engineerとして働くチャンスがあります。英語力はTOEIC800点レベルです。挑戦すべきでしょうか？
```

### 20. 緊急性の高い相談
```
明日重要なプレゼンがあります。緊張して眠れません。すぐにできるリラックス方法と、プレゼンを成功させるコツを教えてください。
```

## 使い方
1. 上記の相談文をコピーしてアプリに貼り付け
2. 各カテゴリでレスポンスの品質を確認
3. レスポンス時間を測定
4. エラーハンドリングの確認
5. 履歴保存機能の動作確認

## 確認ポイント
- [ ] 各相談に対して3人の専門家からの回答が返ってくるか
- [ ] レスポンスが適切な内容か
- [ ] エラーハンドリングが正しく動作するか
- [ ] 履歴が正しく保存されるか
- [ ] UI/UXが使いやすいか
- [ ] ローディング表示が適切か
- [ ] 通信エラー時の挙動が適切か
