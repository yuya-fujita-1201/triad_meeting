# Apple 標準価格帯（日本円・サブスクリプション用）

Appleはサブスクリプション商品に対して固定の価格帯を提供している。
任意の金額は設定できない。以下は日本円での主な価格帯（2026年2月時点）。

## 低価格帯（¥100〜¥1,000）

| JPY | USD相当 |
|-----|---------|
| ¥100 | $0.49 |
| ¥200 | $0.99 |
| ¥300 | $1.49 |
| ¥400 | $1.99 |
| ¥500 | $2.49 |
| ¥600 | $2.99 |
| ¥700 | $3.49 |
| ¥800 | $3.99 |
| ¥900 | $4.49 |
| ¥1,000 | $4.99 |

## 中価格帯（¥1,000〜¥5,000）

¥1,000以降は概ね¥100〜¥200刻みで増加するが、全ての100円刻みが存在するわけではない。
正確な一覧はApp Store Connectの価格設定画面で確認すること。

## よくある失敗パターン

- **¥380** → 存在しない。**¥400**を使う
- **¥480** → 存在しない。**¥500**を使う
- **¥980** → 存在しない可能性あり。App Store Connectで要確認
- **¥1,480** → 存在しない可能性あり。App Store Connectで要確認

## 対策

企画段階で価格を決める際は、Apple標準価格帯を事前に確認すること。
「¥X80」系の価格（¥380, ¥480, ¥980等）は日本のアプリ市場で一般的な感覚だが、
Appleの価格帯には存在しないことが多い。

最寄りの利用可能な価格帯に合わせるか、最初からApple価格帯を前提に企画すること。
